# Portfolio Strategist

AI時代の資産形成において、最も重要なのは「アセットアロケーション（資産配分）」の維持です。
しかし、リバランスのために資産を売却すると、その都度利益に対して課税され、複利効果を阻害してしまいます。

Ayato Portfolio Strategist は、**「売却しないリバランス」** を実現するための専用ツールです。

## 特徴

1. **ノーセル・リバランス**: 現在の保有資産を維持したまま、目標比率に到達するために「次に何を買えばいいか」を正確に計算します。
2. **ハイブリッド管理**: 主要なインデックス投資信託（S&P500, オルカン等）に加え、VTや個別株、暗号資産などを自由にカスタム追加できます。
3. **エッジ完結型プライバシー**: 入力されたデータは一切サーバーに送信されません。あなたのブラウザ内（Local Storage）でのみ処理されます。
4. **直感的なビジュアライズ**: 現状と目標の乖離をチャートで一目で把握できます。

## 使い方

1. 現在保有している各資産の金額を入力します。
2. 目標とするアセットクラスごとの比率を設定します（デフォルト設定も利用可能）。
3. 「Generate Strategy」をクリックすると、具体的な購入プランが表示されます。
