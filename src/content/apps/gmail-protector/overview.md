# Gmail Protector

Gmail Protector は、Google の最新 AI である Gemini API を活用し、メール送信時のセキュリティを飛躍的に向上させる Chrome 拡張機能です。

## 主な機能
- **リアルタイムスキャン**: 送信ボタンを押す直前に、メール本文と添付ファイルを AI が自動解析。
- **機密情報の検出**: 住所、電話番号、パスワード候補、社外秘情報などの漏洩リスクを警告。
- **Google Drive 連携**: 添付された Drive ファイルの中身もチェック可能。
- **プライバシーファースト**: データはローカルで処理され、AI解析時以外に外部サーバーへ送信されることはありません。

## はじめに
1. 拡張機能をインストールします。
2. 設定画面から Gemini API キーを設定します。
3. Gmail を開くと、送信時に自動的にセキュリティチェックが走るようになります。
